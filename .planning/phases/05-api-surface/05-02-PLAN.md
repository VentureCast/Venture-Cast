  executeOrder({ userId, marketId, side:'buy'|'sell', qty?, cashCents?, idempotencyKey,

    -> { trade, order, marketState?, replayed? }
    throws ExecutionError (.statusCode 400/404/409) / RiskError (.statusCode). Replay returns the

  Also exports ExecutionError.

services/amm/execution/priceTrade.js:
  priceTrade(market, marketState, side, { qty } | { cashCents })
    -> { side, deltaQty, grossCents, spreadCents, feeCents, totalCents?(buy), netCents?(sell),
         avgPriceCents, endPriceCents, newSupply, newReserveCents }. Pure, no DB. Throws ExecutionError.
  market needs { P0_cents, k_num, k_den, spreadBps, feeBps }; marketState needs { supply,
  reserveCents, reserveFloorCents }.

services/amm/pricing/curve.js: priceCents(s, {P0_cents,k_num,k_den}) -> integer cents.

LedgerAccount accountKey conventions (models/LedgerAccount.js):
  user_cash:<uid> (cents), user_pos:<uid>:<marketId> (shares), market_reserve:<marketId> (cents).

models/Trade.js fields include { _id, marketId, userId, side, qty, grossCents, feeCents,
  spreadCents, netCents, avgPriceCents, supplyBefore, supplyAfter, createdAt }.

middleware/rateLimiters.js: tradeLimiter (apply per-route on POST /quotes + /orders).

fixtures: generateAuthToken(userId). ammFixtures: createTestMarket() -> { market, marketState }.

       - Return { quoteId: require('crypto').randomUUID(), marketId, side, qty: priced.deltaQty,
